# DeepTrace - Cyber Intelligence Platform

DeepTrace is an AI-powered system for detecting coordinated influence campaigns and disinformation networks.
Built for the INNOVIT 2026 Challenge.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Language**: TypeScript
- **Animations**: Framer Motion
- **Visualization**: Recharts
- **Icons**: Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

## Project Structure
- `src/app`: Page routes (Landing, Login, Dashboard)
- `src/components`: Reusable UI components
- `src/components/ui`: Base primitives (Button, Card)
- `src/components/dashboard`: Dashboard specific widgets

## Features
- **Cyber-Intelligence Theme**: Dark mode, glassmorphism, neon accents.
- **Interactive Dashboard**: Real-time charts and risk monitoring.
- **Campaign Explorer**: Detailed analysis of bot networks (Anti-India Narratives).
- **Responsive Design**: Fully mobile-compatible.
